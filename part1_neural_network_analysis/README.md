# Neural Network Analysis Project

This project uses a feed-forward neural network to analyze customer churn data.

## Tasks Completed
- Dataset understanding
- Data preprocessing
- Neural network model building
- Training and evaluation
- Hyperparameter experimentation
- Final reflection

## Tools Used
- Python
- Pandas
- TensorFlow/Keras
- Scikit-learn
- Matplotlib

## Project Goal
The goal of this project is to build and evaluate a neural network model for customer churn prediction.
